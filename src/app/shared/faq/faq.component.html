Preguntas Frecuentes

1. ¿Cómo puedo recuperar la contraseña?

La contraseña se puede recuperar, dando clic en el link "¿Tiene problemas para ingresar?" en la ventana del login. Para más información ingrese al siguiente tutorial.


2. ¿Cómo puedo verificar el estado de una contingencia?

Primero se debe iniciar sesión, luego cuando le de clic en el boton ingresar, le aparecerá la listada de contingencias realizadas, las contingencias que tienen una X son las rechazadas y las que tienen un chulo son las aprobadas. Para más información ingrese al siguiente tutorial.


3. ¿Cómo puedo ver mi perfil o ver las preguntas frecuentes?

En la parte superior derecha, puede encontrar el icono de preguntas frecuentes y Mi Perfil. En el siguiente tutorial te lo explicamos mejor.


4. ¿Cómo puedo Iniciar Sesión?

En el siguiente totorial, se explicará muy detalladamente todo lo que tiene que ver con el inicio de sesión.


5. ¿Cómo puedo manejar de forma rapida la aplicación?

En el siguiente totorial, se explicará muy detalladamente el funcionamiento de la aplicación de una forma sencilla y rapida.


6. ¿Cómo puedo enviar una contingencia?

Para enviar una contingencia, se debe ingresar a contingencias y darle clic en el boton perdir ayuda en contingencia. Para más información, ver el siguiente tutorial.
